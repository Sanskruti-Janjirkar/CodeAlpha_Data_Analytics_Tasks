# Task 2 - Exploratory Data Analysis

 Best Selling Books 2023 

## Project Description
This project performs Exploratory Data Analysis (EDA) on the Best Selling Books 2023 dataset.  
The analysis explores ratings, prices, genres, and relationships between different features.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Most books have ratings between 4.3 and 4.8.
- Action and Self-Improvement genres are most popular.
- Price increases with print length.
- Ratings are not affected by book price.
