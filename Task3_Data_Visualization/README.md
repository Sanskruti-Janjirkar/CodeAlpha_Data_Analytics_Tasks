# Task 3 - Data Visualization
This folder contains all files related to Task 3 of CodeAlpha Data Analytics Internship.

## Dataset
Best Selling Books 2023

## Visualizations Performed
- Price Distribution Histogram
- Rating Distribution Histogram
- Genre Popularity Bar Chart and Pie Chart
- Price vs Rating Scatter Plot
- Line Graph for Publishing Trend
- Correlation Heatmap
- Mini Dashboard

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Most books have high ratings (above 4.4)
- Book price does not strongly affect rating
- Longer books tend to be more expensive
- Certain genres dominate best sellers
